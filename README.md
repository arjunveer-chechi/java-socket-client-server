# Java Client-Server Socket Program

## 📌 Description
This project demonstrates basic TCP client-server communication using Java sockets.  
It implements a simple request-response model where a server listens for incoming client connections and communicates using input/output streams.

## 🚀 Features
- Accepts client connections using ServerSocket
- Establishes communication using Socket
- Implements request-response interaction
- Uses BufferedReader and PrintWriter for data exchange

## 🛠️ Technologies Used
- Java
- TCP Socket Programming
- Client-Server Architecture
- Java I/O Streams

## ⚙️ How It Works
1. The server starts and listens on a specific port.
2. The client connects to the server using the IP address and port.
3. Messages are exchanged between client and server.
4. The server processes the request and sends a response.
